Inventory Integrity Audit

A professional-grade SQL solution for tracking warehouse stock levels and discrepancies. JOINs compare system records against physical counts, CHECK constraints enforce integrity, and a database trigger logs every change automatically.

Full case study GitHub Gist
-- Audit Report: system vs. physical SELECT i.part_number, i.qty_system, a.qty_physical, (a.qty_physical - i.qty_system) AS variance FROM inventory i INNER JOIN audit_log a ON i.part_id = a.part_id WHERE i.qty_system != a.qty_physical;
