📦 Inventory Integrity Audit

Portfolio Module 1: SQL JOINs & Database Automation

Project Overview

A professional-grade SQL solution for tracking warehouse stock levels and discrepancies. This project demonstrates advanced database techniques including JOINs for data validation and triggers for automated audit trails.

Components

📋 Inventory Table

System of record for part numbers and quantities with data integrity checks.

📝 Audit Table

Captures physical counts to identify mismatches between system records and actual inventory.

⚙️ Automation

Uses a Database Trigger to log every change automatically without manual intervention.

How It Works

The system uses a JOIN to compare system records against manual counts, while the TRIGGER ensures an audit trail exists without requiring manual log entry. This is a foundational pattern for Database Administration (DBA).

Key Queries

  • Audit Report: Highlights items where system vs. actual counts don't match
  • Hot List: Identifies the Top 5 parts with the most frequent audit mismatches
  • Discrepancy Analysis: Calculates the variance between expected and physical quantities
